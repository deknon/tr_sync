คู่มือติดตั้ง TRCloud Sync Browser

สำหรับสมาชิกทีม — ไม่จำเป็นต้องรู้จัก Git มาก่อน
Windows 10 / 11

สรุปขั้นตอนทั้งหมด (ทำครั้งแรกครั้งเดียว)

ขั้นที่ 1 ติดตั้ง Python 3.12+
ขั้นที่ 2 ติดตั้ง Git
ขั้นที่ 3 Clone โปรเจกต์ (ดาวน์โหลดโค้ด)
ขั้นที่ 4 ติดตั้ง Playwright
ขั้นที่ 5 ตั้งค่า Gmail (config.json)
ขั้นที่ 6 Setup session (Login ครั้งแรก)
ใช้งานปกติ เปิด 5_sync_menu_browser.bat — จบ
อัตโนมัติ ตั้ง Task Scheduler รัน 6_scheduled_sync.bat ทุกคืน

1 ติดตั้ง Python

  • ดาวน์โหลด Python 3.12 จาก python.org/downloads
  • ตอนติดตั้ง ติ๊ก "Add python.exe to PATH" ก่อนกด Install
  • เปิด Command Prompt แล้วตรวจสอบ:
python --version
✅ ผลที่ถูกต้อง แสดง Python 3.12.x หรือสูงกว่า

2 ติดตั้ง Git

  • ดาวน์โหลด Git จาก git-scm.com/download/win
  • ติดตั้งแบบ Next → Next → Finish ได้เลย ไม่ต้องเปลี่ยน option ใดๆ
  • ตรวจสอบใน Command Prompt:
git --version
✅ ผลที่ถูกต้อง แสดง git version 2.x.x

3 Clone โปรเจกต์ (ดาวน์โหลดโค้ด)

  • เปิด Command Prompt ที่โฟลเดอร์ที่ต้องการเก็บโปรเจกต์ เช่น C:\Tools\
  • รันคำสั่ง:
git clone https://github.com/deknon/tr_sync.git
  • จะได้โฟลเดอร์ tr_sync มาทันที ไม่ต้องทำอีก
  • เข้าโฟลเดอร์:
cd tr_sync
⚠ ครั้งแรกอาจถาม GitHub username/password ใส่บัญชี GitHub ของตัวเอง หรือขอ token จากหัวหน้าทีม

4 ติดตั้ง Playwright (Browser Engine)

  • ใน Command Prompt ที่โฟลเดอร์โปรเจกต์ รัน 2 คำสั่งนี้ตามลำดับ:
pip install playwright
playwright install chromium
ℹ ทำแค่ครั้งแรกครั้งเดียว รอประมาณ 2–5 นาที ระหว่างดาวน์โหลด chromium

5 ตั้งค่า Gmail แจ้งเตือน

  • Copy ไฟล์ template:
copy config.example.json config.json
  • เปิด config.json แล้วแก้ 3 บรรทัด:
{
  "gmail_sender":   "your_email@gmail.com",
  "gmail_password": "xxxx xxxx xxxx xxxx",
  "gmail_receiver": "receiver@gmail.com"
}
⚠ gmail_password คือ App Password ไม่ใช่ password ปกติ Google Account → Security → 2-Step Verification → App passwords → สร้างใหม่ → copy 16 ตัวอักษร
ℹ ส่งหลายคนได้ "gmail_receiver": "a@gmail.com,b@gmail.com"

6 Setup Session (Login ครั้งแรก)

  • รันคำสั่ง:
python trcloud_sync_browser.py --setup
  • Browser Chromium จะเปิดขึ้น — Login TRCloud ด้วย username/password ของตัวเอง
  • Login สำเร็จแล้ว กลับมาที่ Command Prompt แล้วกด Enter
  • จะมีไฟล์ trcloud_session.json ถูกสร้าง — เก็บ cookie login ไว้
🔒 ห้าม copy trcloud_session.json ข้ามเครื่อง ต้องรัน --setup ใหม่ทุกเครื่อง เพราะ session ผูกกับบัญชีของแต่ละคน

✓ ใช้งานประจำวัน

Manual — เลือกวันที่เอง ดับเบิ้ลคลิก 5_sync_menu_browser.bat
โปรแกรมจะดึงโค้ดใหม่จาก GitHub ให้อัตโนมัติทุกครั้ง
  • เลือก Function: O = ORDER / R = RV / A = ทั้งคู่
  • เลือก Platform: X = ทั้งหมด / S = Shopee / T = TikTok / L = Lazada
  • กรอก Start date และ End date (รูปแบบ YYYY-MM-DD เช่น 2026-04-19)
  • เลือก Visible: N = headless (เร็ว) / V = ดู browser (debug)
Auto — รันอัตโนมัติทุกคืน (D-1) ดับเบิ้ลคลิก 6_scheduled_sync.bat หรือตั้ง Task Scheduler
คำนวณวันที่เองอัตโนมัติ รัน ORDER + RV แล้วส่ง email แจ้งผล 1 ฉบับ

+ ฟังก์ชันเพิ่มเติม

Status Sync — อัพเดท status order ย้อนหลัง (Step 1 only) ดึง status delivered / returned / cancelled จาก marketplace สำหรับ order เก่า
วันที่: คำนวณอัตโนมัติ D-14 → yesterday ทุก platform 
python trcloud_sync_browser.py --status
# ปรับจำนวนวัน
python trcloud_sync_browser.py --status --lookback 30
Return Item Sync — Shopee เท่านั้น (Step 6) Filter SH Status = TO_RETURN + RETURNED → Select All → RETURNED ITEMs
ต้องกรอก date range (หรือ interactive prompt) 
python trcloud_sync_browser.py --return-item
# ระบุวัน
python trcloud_sync_browser.py --return-item --start-date 2026-04-01 --end-date 2026-04-20

! ปัญหาที่พบบ่อย

อาการ สาเหตุ วิธีแก้
Session หมดอายุ / เด้งหน้า login Cookie หมดอายุ python trcloud_sync_browser.py --setup
Browser ไม่เปิด Chromium หาย playwright install chromium
python ไม่รู้จัก Python ไม่ได้ add to PATH ติดตั้ง Python ใหม่ ติ๊ก Add to PATH
[git] Pull skipped ยังไม่ได้ login GitHub รัน git pull ด้วยมือ 1 ครั้งแรก
Shop sync ล้มเหลว หน้าเว็บช้า / timeout รันซ้ำด้วย --shop <id> เฉพาะ shop นั้น
ไม่ได้รับ email แจ้งผล config.json ไม่มีหรือ password ผิด ตรวจ config.json — ใช้ App Password ไม่ใช่ password ปกติ
[git] Pull skipped ยังไม่ได้ login GitHub รัน git pull ด้วยมือ 1 ครั้งแรก แล้ว login
